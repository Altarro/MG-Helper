# Checklisty: session prep i in-session use

## Session prep (przed sesja)

- [ ] Sprawdz, czy aktywna paczka generatora ma komplet tabel systemowych.
- [ ] Uzupelnij 5-10 nowych wpisow pod aktualny watek sesji.
- [ ] Przetestuj po jednym losowaniu: postac, lokacja, event, custom.
- [ ] Zweryfikuj tagi osiowe (np. klimat, frakcja, lokacja).
- [ ] Wykonaj backup JSON po wiekszych zmianach.

## In-session use (w trakcie sesji)

- [ ] Przy improvie odpal `Losuj` i od razu zdecyduj: notatka albo encja.
- [ ] Oznacz wynik tagiem sceny (`scena-1`, `scena-2`, ...).
- [ ] Dla NPC zdecyduj, czy przypinac do sceny natychmiast.
- [ ] Przy wielokrotnych probach wlacz `Bez powtorzen`.
- [ ] Na koniec sesji sprawdz historie losowan i zachowaj warte rozwijania pomysly.
