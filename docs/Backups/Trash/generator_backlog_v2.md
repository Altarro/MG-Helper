# Backlog v2 advanced generation

## Kierunki funkcjonalne

- Generator lancuchow zdarzen (sekwencje 2-5 krokow).
- Cross-table constraints (np. zgodnosc tagow lokacji i zdarzen).
- Tryb "scene kit" (NPC + lokacja + event w jednym rollu).

## Integracje

- Import paczek z marketplace (plik + podpis wersji).
- Share paczki miedzy kampaniami (kopiuj jako nowa wersja).
- Eksport anonimowych telemetry insightow (opt-in).

## AI i jakość danych

- Asystent poprawy tabel (wykrywanie duplikatow i luk tematycznych).
- Auto-normalizacja tagow na bazie slownika kontrolowanego.
- Sugestie brakujacych wpisow per typ tabeli.

## UX zaawansowany

- Tryb "live deck": karta wynikow do szybkiego reveal przy stole.
- Makra MG (wielokrotne rollowanie z predefiniowanym presetem).
- Widok trendow uzycia tabel per sesja.
