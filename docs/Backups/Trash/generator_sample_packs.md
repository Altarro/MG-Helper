# Wzorcowe paczki danych do pobrania

Pliki wzorcowe znajduja sie w katalogu `docs/samples/generator/`.

## Dostepne pliki

- `city-intrigue-pack.json` ‚Äî pelna paczka JSON do importu.
- `rumors-table.csv` ‚Äî przyklad importu CSV pojedynczej tabeli.
- `npc-nicknames.csv` ‚Äî szybka tabela przydomkow.

## Jak uzyc

1. Otworz `Ustawienia -> Ustawienia generatora`.
2. Wybierz import `JSON` albo `CSV`.
3. Wczytaj wybrany plik.
4. Dla testu wykonaj losowanie w panelu `Inspiracje`.
