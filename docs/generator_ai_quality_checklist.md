# Checklista jakości paczek AI

- [ ] Czy `pack.description` zawiera wszystkie wymagane keywordy kontraktu?
- [ ] Czy tagi sa spojne miedzy tabelami powiazanymi (`locationType` i `locationName`)?
- [ ] Czy osie tematyczne maja pokrycie (np. miasto, polityka, mrok, port)?
- [ ] Czy nie ma luk w wariantach nazw (male/female/unisex) tam, gdzie wymagane?
- [ ] Czy wagi sa sensowne (brak skrajnych dysproporcji bez uzasadnienia)?
- [ ] Czy wpisy nie zawieraja duplikatow i pustych wartosci?
- [ ] Czy format JSON jest zgodny z kontraktem i nie zawiera markdownu?
