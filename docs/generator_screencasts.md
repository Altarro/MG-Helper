# Screencasty i gify flowow krytycznych

Ten dokument zbiera scope nagran, ktore powinny byc utrzymywane razem z modułem generatora.

## Flowy krytyczne do nagrania

1. Import JSON paczki + losowanie w `Inspiracje`.
2. Losowanie `Postac` -> `Utworz encje` -> `Wstaw do sceny`.
3. Losowanie `Event table` -> `Dodaj do notatki sesji`.
4. Rollback danych generatora z backupu migracyjnego.

## Standard nagran

- Dlugosc: 20-60 sekund.
- Rozdzielczosc: minimum 1280x720.
- Bez dzwieku (napisy w overlay lub opis pod gifem).
- Nazwa pliku: `generator-flow-<slug>-v<nr>.gif/mp4`.
