# Jak uzywac panelu Inspiracje przy stole

## Szybki flow

1. Otworz `Sesja live` i sekcje `Inspiracje`.
2. Wybierz tryb (`Postac`, `Lokacja`, `Tabela zdarzen`, `Tabela wlasna`).
3. Kliknij `Losuj`.
4. Zdecyduj:
   - `Kopiuj wynik`,
   - `Dodaj do notatki sesji`,
   - `Utworz encje z wyniku`.

## Tryby

- `Postac`: laczy imie + przydomek + nazwisko.
- `Lokacja`: laczy typ + nazwe.
- `Tabela zdarzen`: pojedynczy event do sceny.
- `Tabela wlasna`: customowe tabele kampanii.

## Skróty

- `R`: losuj
- `Q`: losuj ponownie

Uwaga: skroty sa ignorowane w polach edycji tekstu.

## Dobre praktyki MG

- Ustaw tag `improv` lub `scena-<nr>` przy szybkim tworzeniu notatek.
- Dla NPC od razu decyduj o opcji `Wstaw do sceny teraz`.
- Gdy testujesz wiele wariantow, wlacz `Bez powtorzen`.

## Po sesji

- Przejrzyj historie losowan.
- Wyeksportuj paczke generatora JSON, jesli mocno zmieniles tabele.
