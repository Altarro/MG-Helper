# Session Live Archive

Kopia kodu widoków sesji wykonana przed redesignem UI/UX.

Zakres archiwum:

- `components/` — komponenty widoków sesji, live, cleanup i raportu
- `hooks/` — hooki sesji używane przez widoki
- `utils/` — narzędzia sesji live i sygnałów
- `types.ts` oraz `index.ts`

Archiwum jest tylko punktem odniesienia do starego wyglądu i logiki prezentacji.
